Top 5 Open-source NestJS Boilerplates to Kickstart Your Project

Explore the best free Boilerplates for Rapid Development with NestJS

Introduction

This boilerplate is the first free open-source option available on GitHub. It's designed for rapid development with NestJS, and includes a setup for GraphQL, TypeORM, and JWT authentication.

1. Brocode NestJS REST API Boilerplate

This boilerplate is the first free open-source option available on GitHub. It's designed for rapid development with NestJS, and includes a setup for GraphQL, TypeORM, and JWT authentication.

Authentication:

  • Sign-in/Sign-up: Yes.
  • Social Sign-in: Yes.
  • Roles: Yes.
  • Confirm Email: Yes.
  • Forgot Password: Yes.

Config Service: A comprehensive service, allowing environment-specific settings, which is essential when working with a NestJS Boilerplate. Supports local and AWS S3 file uploads, with extensive options and support in multiple languages.

Database: PostgreSQL | TypeORM | MongoDB for storing data.

Testing: Unit tests set up with Jest.

GraphQL: Integration full setup for GraphQL.

SWAGGER: Integrated API documentation, making it easy to generate documentation using any NestJS boilerplate.

DOCKER: Full Docker support.

2. Awesome NestJS Boilerplate

A complete NestJS boilerplate available for free on GitHub. It offers a clean and tidy setup with various tested backend implementations, crucial for any NestJS boilerplate.

Authentication:

  • Sign-in/Sign-up: Yes.
  • Social Sign-in: No.
  • Roles: Yes.
  • Confirm Email: No.
  • Forgot Password: No.

Config Service: Full environment-specific support, which is beneficial when customizing a NestJS boilerplate. Supports AWS S3 (non-actionable mailing and multiple languages).

Transactions: Supports multiple languages.

Database: PostgreSQL, TypeORM

Testing: Set up for unit and e2e testing with Jest.

SWAGGER: Integrated API documentation, crucial for ensuring a smooth development process with any NestJS boilerplate.

DOCKER: Full Docker support.

3. NextJS Prisma Starter

A free starter template designed to integrate Prisma ORM with NestJS for efficient database management. It's particularly useful for projects requiring a robust database layer with TypeScript support.

Documentation: Comprehensive and up-to-date, providing clear guidance for developers looking to implement a NestJS boilerplate with Prisma.

Authentication:

  • Sign-in/Sign-up: Yes.
  • Social Sign-in: No.
  • Roles: Yes.
  • Confirm Email: No.
  • Forgot Password: No.

Config Service: Full support for environment-specific settings, ideal for customizing a NestJS template.

Database: PostgreSQL, Prisma ORM.

Testing: Unit testing setup.

SWAGGER: Integrated API documentation, essential for developers using a NestJS boilerplate to create well-documented APIs.

4. Squadron NestJS Boilerplate

Another free starter template designed for scalable enterprise-level applications. It's free to use and includes features like role-based access control, JWT authentication, and more.

Documentation: Comprehensive and up-to-date, which can be valuable for developers looking to understand different NestJS boilerplate options.

Authentication:

  • Sign-in/Sign-up: No.
  • Social Sign-in: No.
  • Roles: No.
  • Confirm Email: No.
  • Forgot Password: No.

Config Service: Not supported. Supports mailing with advanced configurations, which can be tailored to specific NestJS boilerplate needs. Also supports multiple languages.

Database: Knex.js for SQL-based databases.

Testing: Knex.js for SQL-based databases.

SWAGGER: Not supported.

DOCKER: Does not support Docker.

Summary

Comprehensive Solutions: Brocode NestJS REST API Boilerplate and Awesome NestJS Boilerplate offer extensive features for scalable applications, including strong authentication, full documentation, and Docker support, making them excellent choices for developers working with the NestJS boilerplate.

Database Management: NextJS Prisma Starter and Next Hackathon Starter are more focused on developers who need Prisma ORM, with full support for database migrations. They are particularly suitable for projects where a NestJS boilerplate with robust database management is required.

Hackathon-Ready: Next Hackathon Starter is minimalist, ideal for quick project launches and hackathons, though it lacks some advanced features like full CI/CD integration and auto-updating dependencies, which might be necessary for more complex NestJS boilerplate projects.
